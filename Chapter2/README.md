## 2.2.1
初始化不是赋值，初始化的含义是创建变量时赋予某一个初始值，而赋值的含义是把对象的当前值赋予，而以一个新值代替。

如果定义变量时没有指定初值，则变量被默认初始化，此时变量被赋予默认值。

如果是内置类型的变量未被显式初始化，它的值由定义的位置决定。定义于任何函数体之外的变量为0。一种例外情况是，定义在函数体内的内置类型变量将不被初始化。

编译器生成的默认构造函数会按照如下规则初始化成员：
- 如果有提供类内初始值，用它来初始化成员。
- 否则，默认初始化该成员。
类类型变量在定义时，如果有提供类内初始值，则会自动调用合成的默认构造函数进行初始化，否则默认初始化该成员。如果没有默认构造函数，则定义该类型对象时必须提供显示初始化式。

[默认初始化](##2.2.1)
    

```C++
class a
{
public:
    int c = 0;
	int b ;
	a() = default;
};

a B;

int main()
{
   
	a A;
	cout << B.b << B.c << endl; //0 
	cout << A.b << A.c << endl; //默认初始化C，未定义变量
}
```
