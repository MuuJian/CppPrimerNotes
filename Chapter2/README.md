## 2.2.1
初始化不是赋值，初始化的含义是创建变量时赋予某一个初始值，而赋值的含义是把对象的当前值赋予，而以一个新值代替。

如果定义变量时没有指定初值，则变量被默认初始化，此时变量被赋予默认值。

如果是内置类型的变量未被显式初始化，它的值由定义的位置决定。定义于任何函数体之外的变量为0。一种例外情况是，定义在函数体内的内置类型变量将不被初始化。

类类型变量在定义时，如果有提供初始值，则会自动调用默认构造函数进行初始化（不论变量在哪里定义）。如果没有默认构造函数，则定义该类型对象时必须提供显示初始化式。
    
自动生成的默认构造函数使用与变量初始化相同的规则来初始化数据成员。

```C++
class a
{
public:
    int c = 0;
	int b ;
	a() = default;
};

a B;

int main()
{
   
	a A;
	cout << B.b << B.c << endl; //0 
	cout << A.b << A.c << endl; //随机值
}
```
