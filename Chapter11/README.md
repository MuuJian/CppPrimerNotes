# Chapter11

## 11.1

|  关联容器|类型  |
|  :----| :----   |
| map{key, value}          |          不可重复且有序       | 
| set{key}                |           不可重复且有序       |
| multimap{key, value} |          可重复且有序      |
| multiset{key}       |    可重复且有序         |
| unordered_map{key, value} |    不可重复且无序|
| unordered_set{key}         |  不可重复且无序|
| unordered_multimap{key, value}  |     可重复且无序|
| unordered_multiset{key}       |    可重复且无序|
