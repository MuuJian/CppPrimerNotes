# Chapter11

## 11.1

|  关联容器类型  |
|  :----   |
| map: map{key, value}    不可重复且有序       | 
| set： set{key}    不可重复且有序              |
| multimap: multimap{key, value} 可重复且有序  |
| multiset: multiset{key} 可重复且有序         |
| unordered_map: unordered_map{key, value} 不可重复且无序|
| unordered_set: unordered_set{key} 不可重复且无序|
| unordered_multimap: unordered_multimap{key, value} 可重复且无序|
| unordered_multiset: unordered_multiset{key} 可重复且无序|
